# Virtualenv

virtualenv is a tool to create isolated Python environments.


## Installation

```sh
[sudo] pip install virtualenv
```

## Create

```sh
virtualenv .venv
```

### Set option for python version

```sh
virtualenv -p python3 .venv
```

## Active

```sh
. .venv/bin/activate
```

## Deactive

```sh
deactivate
```
