# Cookiecutter: Better Project Templates

### How to install

#### On MacOS

```sh
brew install cookiecutter
```

### How to generate your project

I selected a cookiecutter-python-cli template!

```sh
cookiecutter https://github.com/xuanluong/cookiecutter-python-cli
```
