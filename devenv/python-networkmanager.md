# Python NetworkManager

Install python-networkmanager on Ubuntu 16.04

## Prequiresites

```sh
sudo apt-get install libdbus-glib-1-dev libdbus-1-dev
```
> pycario, pygobject 

```sh
sudo apt-get install libgirepogitory1.0-dev
```

## Installation

```sh
pip install python-networkmanager
```
